What is html?

  1. Html stands for hyper-text markup language.
  2. It is used to give structure to the web pages. Basically html helps us to design website.
  3. Html is case insensitive language.
  4. Html elements tell the browser how to display the content. Eg. A paragraph will be enclosed within paragraph elements and tell the browser that it's a paragraph.Basically elements act like labels.

Structure of html document.

  1. The <!DOCTYPE html> declaratiion defines that this document is an html5 document.
  2. The <html> element is the root element of an HTML page.
  3. The <head> element contains meta information about the HTML page.
  4. The <title> element specifes a title for the HTML page which is shownn in the browser's title bar or in the page's tab.
  5. The <body> element defines the document's body and is a container for all the visible contents, such as headings paragraphs, images, hyperlinks, tables, lists, etc.
  6. The <h1> element defines a large heading.
  7. The <p> element defines a paragraph.

What are html elements?

The HTML element is everything from the start tag to the end tag.
Syntax: <tagname> Content goes here </tagname>

What are attribues?

Attributes contain additional pieces of information. Attributes take the form of an opening tag and additional info is placed inside.
Syntax: <tagname atribute="value" >some text </tagname>
